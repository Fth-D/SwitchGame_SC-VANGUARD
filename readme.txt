■ ゲーム概要
------------------------------------------------------------------
　自機「VANGUARD」は画面左側に位置し、右から迫りくる敵編隊を撃ち落として
　いく横スクロール・シューティングです。背景は右から左へ流れ、宇宙空間を
　前進していく疾走感を演出します。

　本作の核となるのは「変形」と「3種の武器」です。自機はロボット形態と
　ファイター形態に変形でき、形態によって移動速度・射撃間隔・弾の発射口が
　変化します。さらに3種類の武器はそれぞれ長所と短所を持ち、状況に応じて
　使い分けることが攻略の鍵になります。

■ 特徴・アピールポイント
------------------------------------------------------------------
　1. 変形システム（ロボット ⇔ ファイター）
　　　－ボタンで機体を2つの形態に変形。ロボット形態は高速機動（移動速度が
　　　速い）、ファイター形態は安定した射撃に向くなど、形態ごとに移動速度・
　　　射撃間隔・弾の発射口が変化します。状況に合わせて変形しながら戦う、
　　　本作ならではの立ち回りが楽しめます。

　2. 個性のある3種の武器（三すくみ）
　　　・バルカン（扇形拡散）：弾が扇状に広がり、群れの敵をまとめて処理。
　　　　　　　　　　　　　　　　単体の硬い敵には火力不足。
　　　・レーザー（直線・貫通）：高速の直線ビーム。命中しても消えず貫通する
　　　　　　　　　　　　　　　　ため硬い敵や密集に強い。発射中は自機の発射口
　　　　　　　　　　　　　　　　のY位置を追従。狙いは絞られる。
　　　・ホーミング（追尾）：発射（飛び出し）→加速（前進）→最寄りの敵を追尾、
　　　　　　　　　　　　　　の3段階で飛翔。狙わなくても当たるが火力は控えめ。
　　　「広く薄く／狭く強く／自動追尾」で役割を分け、どれか1つが最強に
　　　ならないよう各武器に弱点を設定しています。

　3. リペアコア方式の耐久システム
　　　自機はHPを持ち、被弾でHPが減少。HPが尽きるとリペアコアを1つ自動消費
　　　してHP全回復＋一定時間の無敵を得ます。リペアコアも尽きるとゲーム
　　　オーバーになります（昔ながらのアーケード方式：セーブ・コンティニュー
　　　なし）。

　4. 三層パララックス背景
　　　遠景・中景・近景の3レイヤーをそれぞれスプライトシートアニメーション
　　　で動かし、奥行きと宇宙空間の臨場感を演出しています。

　5. Switch の振動フィードバック
　　　敵と接触した際に Joy-Con を振動させ、被弾を体で感じられるようにして
　　　います。

　6. HUD 描画順の安定化設計
　　　オブジェクト描画ループの後にUI（HPゲージ・フレーム）を描画することで、
　　　オブジェクトの生成・削除順に左右されず、HUDを常に最前面へ安定表示
　　　しています（HPゲージは19コマのスプライトシートアニメで表現）。
